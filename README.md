# NLP Text Summarization

An NLP project focused on abstractive and extractive text summarization techniques using state-of-the-art models.

## Features

- **Abstractive Summarization**: Generates new sentences to capture the main idea of the text.
- **Extractive Summarization**: Identifies and extracts the most important sentences from the original text.
- **Pre-trained Models**: Utilizes popular pre-trained models like BART, T5, and Pegasus for summarization tasks.
- **Evaluation Metrics**: Includes ROUGE and BLEU scores for evaluating summarization quality.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Madid1976/nlp-text-summarization.git
   cd nlp-text-summarization
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Explore the `notebooks/` directory for examples and tutorials on different summarization techniques.
